## ğŸš€ Getting Started

### Prerequisites

- Python 3.x
- pip (Python package installer)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Satyajeet1718/Data-Visualizer.git
cd Data-Visualizer
```

2. Install required dependencies:
```bash
pip install matplotlib pandas
```

### Usage

1. Run the application:
```bash
python app.py
```

2. Choose your data input method:
   - **Manual Entry**: Click "Enter Data Manually" to input your data points
   - **File Upload**: Click "Load Data from File" to import a CSV file

3. Select your graph type from the dropdown menu (Bar Graph, Histogram, Line Graph, or Scatter Plot)

4. Choose your X and Y axis columns

5. Click "Plot Graph" to visualize your data

6. Use "Save Graph" to export your visualization
